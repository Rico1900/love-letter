# AGENTS.md

## 项目目标
- 这是一个写给妻子的 LaTeX 情书项目。
- 当前 `main.tex` 为英文草稿，目标是转换为自然、真诚、可直接排版的中文版本。

## 编辑范围
- 优先只改正文文案，不破坏现有版式宏和装饰元素。
- 非必要不改导言区（字体、颜色、页边距、ornament 宏等）。
- 保留以下结构：
  - 标题（`\lettertitle{...}`）
  - 日期区块（`\today`）
  - 正文区块（两条 `\ornamentrule` 之间）
  - 署名区块

## 中文改写规范
- 语气：亲密、克制、真诚，避免空泛“鸡汤”。
- 文风：简洁，句长适中，朗读顺滑。
- 标点：使用中文全角标点（，。；：？！“”）。
- 占位符处理：
  - `[Her Name]` 可保留，或替换为具体称呼（按用户要求）。
  - `[Your Name]` 同上。
- 不随意新增段落数量，尽量与原英文段落一一对应，方便校对。

## LaTeX 约束
- 继续使用 `ctex`，默认采用 `xelatex` 编译。
- 不引入额外外部资源文件（图片、字体文件）除非用户明确要求。
- 如遇中文换行或标点问题，优先通过文案微调解决，避免大改版式。

## 验收标准
- 能成功编译（至少两遍 `xelatex`）且无报错。
- PDF 中中文显示正常，无乱码、无明显溢出。
- 情感表达与英文原稿语义一致，不丢失核心承诺和感谢段落。
